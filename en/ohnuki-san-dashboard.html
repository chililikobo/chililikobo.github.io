Ohnuki-san Lite or Plus is Required

Ohnuki-san Dashboard is a viewer designed to load and analyze data exported by Ohnuki-san Lite or Ohnuki-san Plus.

Since this tool does not collect system information on its own, please prepare a data folder (report set) generated by either of these tools beforehand.
*Ohnuki-san Lite is a streamlined version of the Plus edition with a limited set of collection items. For comprehensive data visualization, please refer to the comparison table and consider using Ohnuki-san Plus.

Product Description

"Turning massive reports into instant insights."

Ohnuki-san Dashboard was developed to integrate scattered system information into one place, providing a graphical overview of your IT asset status.

Looking for a data collection tool?

To create the HTML reports required for analysis, please use "Ohnuki-san Plus," our system information extraction tool.
