Product Overview

"All the essential info, in the smallest possible space."

Simple Stats is a real-time system monitor that displays PC resource status and network information in a compact window.

Featuring a high-visibility dark theme and "Always on Top" capability, it lets you keep an eye on your system without interrupting your workflow.

Key Features

  • Real-time Monitoring: Visualizes CPU, Memory, Disk, and Network load with gauges.
  • Network Information: Displays Hostname, IP Address, Subnet, Gateway, and DNS.
  • Smart Design: Auto-saves window position and releases memory when minimized.
  • Alert System: Gauge colors change to notify you when load exceeds 80%.
